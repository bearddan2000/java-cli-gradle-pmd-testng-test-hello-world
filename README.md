# java-cli-gradle-pmd-testng-test-hello-world

## Description
Analyze source code for potential bugs.
Analyze source code for potential bugs.
A POC for gradle app using TestNG.
Also dumps jacoco-findbugs reports
to the reports folder.

## Tech stack
- java
- gradle
	- pmd
	- spotbugs
  - TestNG
  - jacoco

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
